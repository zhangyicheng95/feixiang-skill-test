# 数学公式与函数类 HTML 生成技能规范

## 一、 视觉硬性要求 (Visual Identity)

1. **一屏展示**：所有内容必须在 16:9 画布内完成，严禁出现滚动条。
2. **禁止 Emoji**：全篇不得出现任何表情符号，保持学术严谨性。
3. **卡片设计**：
   - 1px 描边：`border: 1px solid ${activeColor}4D;`（4D 代表 30% 透明度）。
   - 描边圆角：`border-radius: 16px;`。
   - 禁止填充：卡片背景必须透明，不带任何背景色。
4. **字体选择**：使用几何感强的无衬线字体（如 Inter, Roboto, PingFang SC）。
