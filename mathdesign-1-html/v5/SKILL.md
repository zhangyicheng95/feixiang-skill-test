# 数学公式与函数类 HTML 生成技能规范

## 一、 视觉硬性要求 (Visual Identity)
1. **一屏展示**：所有内容必须在 16:9 画布内完成，严禁出现滚动条。使用 `height: 100vh; overflow: hidden;` 约束容器。
2. **禁止 Emoji**：全篇严禁出现任何表情符号，确保学术严谨感。
3. **公式渲染**：必须使用 MathJax 渲染 LaTeX。
4. **透明卡片设计**：
   - 1px 描边：`border: 1px solid ${primaryColor}4D;`（4D 代表 30% 透明度）。
   - 描边圆角：`border-radius: 16px;`。
   - 禁止填充：`background: transparent;`（严禁背景色块，保持视觉通透）。
5. **字体选择**：几何感无衬线体（Inter, Roboto, PingFang SC）。

## 三、 布局与代码约束 (防留白逻辑)
1. **比例化布局**：强制使用 `display: grid; grid-template-columns: 2fr 1fr;`。左侧主展示区必须占据 60%-70% 宽度，右侧控制/说明区占据 30%-40% 宽度。
2. **垂直撑满**：右侧卡片区必须使用 `display: flex; flex-direction: column; justify-content: space-between;`，确保卡片内容在垂直方向上均匀分布，消除底部空白。
3. **SVG 动态化**：所有几何图形必须使用 `<svg>` 绘制，并为关键元素添加 CSS 动画（如 `transition` 或 `animate`）。
4. **全局间距**：设置 `padding: 24px; gap: 24px;` 确保呼吸感。
