# 验收阶段 · 调用流程图（开发者文档）

> 路径：`test-html/只供用户查看的调用流程.md`  
> **⚠️ 仅供开发者阅读，Agent 执行时不要 Read 本文件。** 实际调用以 `test-html/SKILL.md` 为准。

---

## 总览

```mermaid
flowchart TD
    START([Step 2 完成]) --> ENTRY[Read test-html/SKILL.md]
    ENTRY --> MUST[抽 must-cover 清单]
    MUST --> TEST[调用 test_html / Playwright]
    TEST --> CARD[输出验证结论卡]
    CARD --> OK{状态=✓ 且 N/M 全中?}
    OK -->|是| DONE([可宣称交付])
    OK -->|否| BACK{产物类型?}
    BACK -->|单页| FIX_S[回 html-authoring Step 2]
    BACK -->|多页| FIX_M[回 courseware-generator Step 2]
    FIX_S --> ENTRY
    FIX_M --> ENTRY
```

---

## Playwright 分支

```mermaid
flowchart TD
    PW_START([Read SKILL.md]) --> MUST[must-cover 清单]
    MUST --> R_TPL[按需 Read references/test-templates.md]
    R_TPL --> CODE[组合 playwrightCode]
    CODE --> RUN[调用 test_html]
    RUN --> FAIL{通过?}
    FAIL -->|是| CARD[验证结论卡]
    FAIL -->|否| DIAG[失败处理见 SKILL.md]
    DIAG --> RUN
```
