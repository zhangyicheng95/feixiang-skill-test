# 多页课件 · 生图提示词增强指南

> 路径：`courseware-generator/references/image-generation.md`  
> **只在已经准备调用 `generate_images` 时读取。** 本指南仅服务本 skill（多页翻页课件），不决定是否生图，不新增图片需求，不替换用户上传图/教材原图/已写入 `artifact-spec.assets` 的 URL。

---

## 一、适用边界与固定流程

```text
模型根据大纲或强互动页需求判断需要真实图片素材
  ↓
形成初始图片描述（主题、主体、用途）
  ↓
若是第 1 页封面图：先读 references/cover.md，定 coverImageSlot
  ↓
逐条判断是否命中第四节「内容类型 → 固定风格」
  ├─ 命中：只给该条追加右侧风格提示词
  └─ 不命中：不套风格库，只保留原始需求 + 通用质量约束
  ↓
调用 generate_images → 拿回真实 URL
  ↓
写入 artifact-spec.assets（url、用途、页码/位置、source、fallback）
```

关键原则：

1. **命中式增强**：按单张图判断，禁止整套课件统一套一种画风。
2. **不改变生图时机**：不能为了套风格而额外新增图片需求。
3. **不改变教学内容**：只能补画风、构图、质感、色彩与质量约束。
4. **非命中不硬套**：数学示意图、公式推导、实验装置结构图、统计图、流程图、UI/按钮图标等，禁止强行工笔/水墨/敦煌/剪纸等。
5. **不替换已有素材**：用户上传、教材原图、题图、已确定 URL，不因本指南重生。
6. **封面先定槽位**：第 1 页封面图无论是否命中风格映射，都必须先按 `references/cover.md` 写清 `coverImageSlot`（`targetPx` / `aspectRatio` / `promptHint`）。
7. **禁止虚构路径**：不得编造 `*_placeholder.png`、相对本机路径、Skill 内部路径；`generate_images` 不可用时改 SVG/CSS/Canvas 自绘，并在 `assets` 声明。

## 二、工具与参数

### 2.1 工具名

本服务部署的生图工具名为 **`generate_images`**（复数）。必须以当前会话工具清单为准；不要臆造 `generate_image`、`picture_gen` 等别名。

| 项 | 说明 |
|---|---|
| 工具名 | `generate_images` |
| 典型参数 | `imageDescriptions: string[]`（每项一条完整 prompt 字符串） |
| 返回 | 真实可访问的图片 URL（如 `*.fbcontent.cn`），写入 `artifact-spec.assets` |

本地开发环境若未挂载 `generate_images`，降级为 SVG/CSS/Canvas 自绘，并在 `assets` 声明原因；**禁止**虚构 URL。

### 2.2 提示词语言

- 豆包 / 中文美术风格场景：**优先中文提示词**。
- 不要把「工笔重彩、水墨写意、敦煌岩彩、剪纸剪影、白描线稿」硬译成失真英文；工具强制英文时，中文风格关键词可中英并列。
- **禁止**要求图片生成可见标题、题目、公式长文、页码、按钮文案；文字由 HTML 排版承载。

### 2.3 与 artifact-spec.assets 的合同

```json
{
  "id": "cover-01",
  "url": "https://...",
  "role": "cover",
  "page": 1,
  "source": "generate_images",
  "core": false,
  "fallback": "CSS gradient + SVG icon",
  "coverImageSlot": {
    "layout": "cover-07-top-title-image-card",
    "slotId": "cover-07-image-card",
    "visual": "inline-card",
    "targetPx": "900x357",
    "aspectRatio": "2.52:1",
    "promptHint": "超宽横向图，主体居中，不含文字/边框"
  },
  "prompt": "最终发给 generate_images 的完整描述",
  "styleHit": "水彩淡彩 | none"
}
```

- 图片/音频 URL 必须真实可访问；**禁止**大体积 base64 音视频嵌入。
- 外部图只能作非核心增强：`core:false`，并提供 `fallback`。
- 生图失败或工具不可用：在 `assets` 写明原因并自绘，**禁止**假 URL 占位。

## 三、封面图槽位约束（高优先级）

封面图调用 `generate_images` 前：

1. 读取 `references/cover.md`，确定 `data-cover-visual`、`data-cover-layout`、`coverImageSlot`。
2. 把 slot 写入 `artifact-spec.assets`。
3. Prompt 写明目标比例、横竖方向、安全构图；禁止「任意比例 + HTML `object-fit:cover` 裁主体」。
4. 非 `full-bleed-background` 时必须写明「局部图片槽位，不是整页背景」。

封面 prompt 必含：

```text
用于互动课件第 1 页封面，目标图片槽位：<slotId>，目标尺寸约 <targetPx>，目标比例 <aspectRatio>，<promptHint>。画面无文字、无标题、无公式、无页码、无边框；主体完整落在槽位安全区内。
```

局部封面额外必写：

```text
这不是整页背景图，而是放入封面局部图片容器的素材；请严格按 <targetPx> / <aspectRatio> 构图，主体完整，不要厚边框，不要自带圆角或留白外框。
```

| slotId | 适用版式 | prompt 必写比例 |
|---|---|---|
| `cover-full-bleed` | cover-01～04 整页背景 | `16:9 横向整页背景，目标 960×540，主体避开文字安全区` |
| `cover-05-left-visual` | 左图右文 | `约 480×540，0.89:1，近方形/轻竖向，不是整页背景` |
| `cover-06-organic-visual` | 左侧异形遮罩 | `约 415×709，0.59:1 竖向主体，适合圆角/胶囊遮罩，主体勿贴边` |
| `cover-07-image-card` | 上文下图卡片 | `约 900×357，2.52:1 超宽横向，禁止竖图` |

页内非封面插图：无需 `coverImageSlot`，仍走第四节命中判断，并在 prompt 中写清用途与构图。

## 四、固定命中映射

每条描述独立判断。命中左侧内容类型时，**必须**用右侧一一对应风格，禁止命中后再自由换风格。

| 左侧内容类型 | 典型图片需求 | 右侧固定风格 |
|---|---|---|
| 古典文学名著 | 宫廷、服饰、器物、古典小说场景 | 工笔/重彩（Gongbi） |
| 古诗词、古代史 | 山水游记、古代文人、历史意境 | 水墨/写意（Ink Wash） |
| 近现代文学 | 革命历史、严肃叙事、近现代小说 | 版画/木刻（Woodblock Print） |
| 神话传说 | 丝路、神话人物、古代艺术史 | 敦煌/岩彩（Dunhuang Mural） |
| 传统文化 | 传统节日、趣味语文、传统元素现代表达 | 新国潮插画（New Guochao） |
| 现代散文、抒情诗 | 景物、校园、亲情友情、抒情场景 | 水彩淡彩（Watercolor Light） |
| 文化交融主题 | 中外交流、跨文化、综合性学习封面 | 拼贴艺术（Collage Art） |
| 民俗文化、人物剪影 | 民俗节日、寓言、群像剧场感 | 剪纸剪影（Paper-cut Silhouette） |
| 古代建筑知识 | 亭台楼阁、纹样、构造认知 | 白描/线稿（Bai Miao / Line Art） |

匹配规则：

1. 先认核心教学对象属于哪一类，再追加该行风格。
2. 禁止先选风格再反向改写图片类型。
3. 多类像时以主要教学目标为准，只选一类。
4. 无法确定则**不命中**，不追加风格库。

不命中示例（只加通用质量约束）：

- 数学几何、数轴、面积模型、公式推导
- 理化生实验装置结构图
- 统计图、流程图、知识结构图
- 按钮/奖励/操作提示图标
- 必须还原教材原图或用户上传图

通用质量约束示例：`主体清晰，构图简洁，无文字，无厚边框，适合对应页面图片区域`；封面须适配 `coverImageSlot`。

## 五、固定追加提示词

命中后只追加对应一条，禁止堆叠多种传统美术风格。

### 5.1 工笔/重彩

```text
工笔/重彩风格，线条细腻工整，色彩艳丽而典雅，朱红、石青等传统设色，人物服饰、器物和场景细节丰富，典雅、华丽、精致，无文字
```

### 5.2 水墨/写意

```text
水墨/写意风格，黑白灰为主，留白充足，笔触飘逸，虚实结合，画面飘逸、空灵、淡远，有古典文学意境，无文字
```

### 5.3 版画/木刻

```text
版画/木刻风格，线条硬朗，黑白对比强烈，刻痕质感明显，画面古朴、沧桑，具有力量感、叙事感和历史厚度，无文字
```

### 5.4 敦煌/岩彩

```text
敦煌/岩彩风格，模仿壁画肌理，矿物色彩斑驳沉稳，带有剥落感，线条流动，画面神秘、古老、有信仰感，无文字
```

### 5.5 新国潮插画

```text
新国潮插画风格，传统元素与现代构图结合，高饱和度色彩，矢量感强，画面时尚、活泼、有趣味，适合课堂课件，无文字
```

### 5.6 水彩淡彩

```text
水彩淡彩风格，色彩轻柔透明，光感温和，画面干净，情绪抒情，适合语文阅读课件，无文字
```

### 5.7 拼贴艺术

```text
拼贴艺术风格，纸张肌理和图形层次，低饱和配色，构图有节奏，画面像文学主题视觉海报，但不包含文字
```

### 5.8 剪纸剪影

```text
剪纸剪影风格，人物和场景轮廓清晰，层次分明，具有传统民间艺术质感，画面简洁有戏剧感，无文字
```

### 5.9 白描/线稿

```text
白描/线稿风格，线条干净准确，主体轮廓清晰，少量淡彩，便于课堂观察和讲解，适合古建筑结构和器物纹样认知，无文字
```

## 六、何时生图 / 何时自绘

| 优先 | 场景 |
|---|---|
| 生图 | 封面氛围图；语文/历史/阅读情境插画；真实景物/人物/器物主视觉；用户明确要求 AI 配图 |
| 自绘 SVG/CSS/Canvas | 数学图、实验结构示意、流程图、UI 图标、精确几何关系 |
| 禁止生图替代 | STEM「原题/原卷」必须保真的图；用户要求一比一还原原图；已有真实 URL |

同一角色或连续场景多张图：保持角色设定、服装、色调一致。

## 七、调用前自检

```text
□ 本条是已判断需要的图，不是为套风格新增
□ 封面图已定 coverImageSlot，并写入 prompt 与 artifact-spec.assets
□ 已做命中判断；命中才追加固定风格，未命中未强行套库
□ 未堆叠多种风格，未自由换映射表外的风格
□ 提示词语言适配当前生图工作流（豆包优先中文）
□ 未要求图片内可见文字/标题/公式/按钮文案
□ 连续角色/场景设定一致
□ 未替换用户上传图、教材原图、已确定 URL
□ generate_images 不可用时已改为自绘并在 assets 声明，无虚构 URL
□ 结果已写入 artifact-spec.assets，source=generate_images
```
