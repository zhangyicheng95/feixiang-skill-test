# 互动课件生图提示词增强指南

> 本文件只在模型已经判断需要调用 `generate_image` 时读取。它不决定是否生图，不新增图片需求，不替换教材/用户已有图片，也不参与模板后处理。

---

## 一、适用边界

当且仅当已经准备调用 `generate_image` 时，先按本指南逐条检查即将传入 `imageDescriptions` 的每一条图片描述。

固定流程：

```text
模型根据大纲或强互动页实现需要，自行判断需要生图
  ↓
模型形成初始图片描述
  ↓
逐条判断该图片是否命中本指南的增强类型
  ├─ 命中：只给这一条图片描述追加对应风格提示词
  └─ 不命中：不追加本指南风格，只保留原始图片需求和必要质量约束
  ↓
调用 generate_image
```

关键原则：

1. **命中式增强**：按单张图片判断，不按整套课件统一套风格。
2. **不改变生图时机**：本指南不能让模型为了套风格而额外生成图片。
3. **不改变教学内容**：增强提示词只能补充画风、构图、质感、色彩和质量约束，不能改变图片主题、人物、场景、知识点或教学目标。
4. **不影响非命中图片**：数学示意图、公式推导图、实验结构图、数据图表、流程图、UI 图标、按钮奖励图标等不命中本指南时，禁止强行追加工笔、水墨、敦煌、剪纸等风格。
5. **不替换已有素材**：用户上传图片、教材原图、题图、已经生成并确定使用的素材 URL，不因为本指南重新生成。
6. **不用于模板注入**：Phase 6 / Phase 7 套模板时禁止调用 `generate_image`，也禁止按模板重新生图、替换图片或改变图片颜色。

---

## 二、提示词语言与工具参数

当前生图链路通过 `generate_image` 工作流接收 `imageDescriptions` 字符串数组。提示词语言按当前生图工作流的最佳理解能力生成；在豆包生图工作流下，**优先使用中文提示词**。

要求：

- 保留参数名 `imageDescriptions`，每一项是一个字符串，不要包装成 `{ "prompt": "..." }`。
- 不要因为旧工具说明中出现英文要求，就把“工笔重彩、水墨写意、敦煌岩彩、剪纸剪影、白描线稿”等中文风格词硬翻译成失真的英文。
- 若工具调用当轮的参数校验明确要求英文，再按工具校验要求调整；但中文传统美术风格关键词可以保留为中文关键词或中英并列，避免语义漂移。
- 图片提示词中禁止要求生成可见中文标题、课件标题、题目文字、公式长文本、页码、按钮文字或英文栏头。文字、公式和标题应由 HTML 排版承载。

---

## 三、命中判断

每条图片描述独立判断。命中以下类型时，才允许追加对应风格提示词。

| 命中类型 | 典型图片需求 | 可追加的风格方向 | 注意 |
|---|---|---|---|
| 语文/历史封面图 | 课文封面、单元封面、诗词封面、历史主题封面 | 工笔重彩、水墨写意、版画木刻、敦煌岩彩、新国潮插画、水彩淡彩、拼贴艺术、剪纸剪影、白描线稿 | 只选一种主风格，禁止堆叠多个风格 |
| 文学意境图 | 古诗词意境、文言文场景、散文情境、人物故事氛围 | 水墨写意、水彩淡彩、白描线稿、工笔重彩 | 画面服务情境，不生成诗句文字 |
| 历史文化图 | 古代人物、历史场景、传统文化、民俗节日、神话传说 | 工笔重彩、敦煌岩彩、新国潮插画、剪纸剪影、版画木刻 | 避免现代服饰、现代建筑和错代器物 |
| 课文人物/场景插图 | 课文人物、故事关键场景、阅读导入场景 | 水墨写意、版画木刻、水彩淡彩、白描线稿 | 同一人物多图时保持年龄、服饰、气质一致 |
| 古建筑/器物知识图 | 亭台楼阁、器物、传统纹样、建筑结构认知 | 白描线稿、工笔线描、淡彩示意 | 若教学目标是结构识别，清晰准确优先于装饰 |

不命中示例：

- 数学几何图、数轴、面积模型、公式推导、计算步骤图。
- 物理/化学/生物实验装置结构图。
- 统计图、流程图、表格、知识结构图。
- 练习题选项、按钮图标、奖励星星、操作提示图标。
- 需要严格还原教材原图或用户上传图片的场景。

这些图片不追加本指南的风格库。若仍需生图，只加入通用质量约束：主体清晰、构图简洁、无文字、适合 16:9 课件画面、不要复杂边框。

## 四、风格增强规则

命中后，只能选一个最贴合的风格方向。不要把“工笔 + 水墨 + 敦煌 + 剪纸 + 新国潮”同时塞进同一条提示词。

### 4.1 工笔重彩

适合：古典文学人物、古代人物、宫廷/历史场景、庄重华丽的封面。

可追加：

```text
工笔重彩风格，线条细致，色彩饱和但典雅，人物服饰和场景细节精致，传统中国画质感，画面庄重，适合语文历史课件封面，无文字
```

### 4.2 水墨写意

适合：古诗词、山水意境、文人场景、抒情导入图。

可追加：

```text
水墨写意风格，留白充足，墨色层次自然，淡彩点染，画面安静含蓄，有中国古典文学意境，无文字
```

### 4.3 版画木刻

适合：近现代文学、劳动场景、群体人物、质朴有力量的历史叙事。

可追加：

```text
版画木刻风格，黑白或低饱和套色，线条有刻痕质感，人物和场景轮廓有力量，质朴叙事感，无文字
```

### 4.4 敦煌岩彩

适合：神话传说、丝路、飞天、壁画感历史文化主题。

可追加：

```text
敦煌岩彩风格，壁画质感，矿物色彩，线条流动，古典纹样点缀，画面有历史文化气息，无文字
```

### 4.5 新国潮插画

适合：传统文化的现代化表达、节日民俗、年轻化封面。

可追加：

```text
新国潮插画风格，传统元素现代化表达，色彩明快但不刺眼，构图简洁，适合课堂课件封面，无文字
```

### 4.6 水彩淡彩

适合：现代散文、自然景物、抒情诗、温柔导入场景。

可追加：

```text
水彩淡彩风格，色彩轻柔透明，光感温和，画面干净，情绪抒情，适合语文阅读课件，无文字
```

### 4.7 拼贴艺术

适合：文化交融、阅读写作、跨材料主题、现代文学综合封面。

可追加：

```text
拼贴艺术风格，纸张肌理和图形层次，低饱和配色，构图有节奏，画面像文学主题视觉海报，但不包含文字
```

### 4.8 剪纸剪影

适合：民俗、节日、群像、寓言故事、传统故事剧场感画面。

可追加：

```text
剪纸剪影风格，红黑或低饱和民俗配色，人物和场景轮廓清晰，层次分明，传统民间艺术质感，无文字
```

### 4.9 白描线稿

适合：古建筑、器物、传统纹样、人物关系简图、需要清晰轮廓的知识图。

可追加：

```text
白描线稿风格，线条干净准确，少量淡彩，主体轮廓清晰，便于课堂观察和讲解，无文字
```

## 五、模板关系

模板选择不参与“是否命中本指南”的判断。

1. `selectedTemplateId` 不能把非命中图片强行变成命中图片。
2. `selectedTemplateId` 不能把语文历史图片强行改成像素风、复古海报风或模板截图风。
3. 模板只要求图片素材具备通用承载性：主体清晰、边缘干净、背景不过度复杂、无自带文字、无厚重边框、适合放入 960×540 课件页面。
4. Phase 7 套模板时只能保护并呈现已有图片 URL，不能重新生图、替换图片、给图片加滤镜或改变图片颜色。

---

## 六、调用前自检

调用 `generate_image` 前，逐条自检：

- [ ] 这条图片是模型已经判断需要生图的图片，不是为了套风格新增的图片。
- [ ] 已判断是否命中本指南；命中才追加风格，不命中没有套风格库。
- [ ] 命中时只选了一个风格方向，没有堆叠多个传统美术风格。
- [ ] 提示词使用当前生图工作流更稳定的语言；豆包工作流下优先中文。
- [ ] 没有要求图片生成可见文字、标题、题目、公式或按钮文案。
- [ ] 同一人物或同一场景的连续图片保持角色设定一致。
- [ ] 没有替换用户上传图片、教材原图或已确定使用的素材 URL。
