# 互动课件生图提示词增强指南

> 本文件只在模型已经判断需要调用 `generate_image` 时读取。它不决定是否生图，不新增图片需求，不替换教材/用户已有图片，也不参与模板后处理。

---

## 一、适用边界

当且仅当已经准备调用 `generate_image` 时，先按本指南逐条检查即将传入 `imageDescriptions` 的每一条图片描述。

固定流程：

```text
模型根据大纲或强互动页实现需要，自行判断需要生图
  ↓
模型形成初始图片描述
  ↓
若该图片是第 1 页封面图：先读取/复用 cover-layout-guide.md 的 coverImageSlot
  ↓
逐条判断该图片是否命中本指南的左侧内容类型
  ├─ 命中：只给这一条图片描述按固定映射追加右侧风格提示词
  └─ 不命中：不追加本指南风格，只保留原始图片需求和必要质量约束
  ↓
调用 generate_image
```

关键原则：

1. **命中式增强**：按单张图片判断，不按整套课件统一套风格。
2. **不改变生图时机**：本指南不能让模型为了套风格而额外生成图片。
3. **不改变教学内容**：增强提示词只能补充画风、构图、质感、色彩和质量约束，不能改变图片主题、人物、场景、知识点或教学目标。
4. **不影响非命中图片**：数学示意图、公式推导图、实验结构图、数据图表、流程图、UI 图标、按钮奖励图标等不命中本指南时，禁止强行追加工笔、水墨、敦煌、剪纸等风格。
5. **不替换已有素材**：用户上传图片、教材原图、题图、已经生成并确定使用的素材 URL，不因为本指南重新生成。
6. **不用于模板替换**：进入 `courseware-template-applier` 后禁止调用 `generate_image`，也禁止按模板重新生图、替换图片或改变图片颜色。
7. **封面图先匹配槽位**：第 1 页封面图无论是否命中第四节风格映射，都必须先按 `cover-layout-guide.md` 的 `coverImageSlot` 写清目标尺寸、比例和构图安全区。

---

## 二、提示词语言与工具参数

当前生图链路通过 `generate_image` 工作流接收 `imageDescriptions` 字符串数组。提示词语言按当前生图工作流的最佳理解能力生成；在豆包生图工作流下，**优先使用中文提示词**。

要求：

- 保留参数名 `imageDescriptions`，每一项是一个字符串，不要包装成 `{ "prompt": "..." }`。
- 不要因为旧工具说明中出现英文要求，就把“工笔重彩、水墨写意、敦煌岩彩、剪纸剪影、白描线稿”等中文风格词硬翻译成失真的英文。
- 若工具调用当轮的参数校验明确要求英文，再按工具校验要求调整；但中文传统美术风格关键词可以保留为中文关键词或中英并列，避免语义漂移。
- 图片提示词中禁止要求生成可见中文标题、课件标题、题目文字、公式长文本、页码、按钮文字或英文栏头。文字、公式和标题应由 HTML 排版承载。

---

## 三、封面图槽位约束（高优先级）

当某条 `imageDescriptions` 用于第 1 页封面图时，必须先完成以下步骤，再判断是否命中风格映射：

1. 读取或复用 `cover-layout-guide.md`，确定 `data-cover-visual`、`data-cover-layout` 和 `coverImageSlot`。
2. 把 `coverImageSlot` 写进素材清单：`slotId`、`targetPx`、`aspectRatio`、`promptHint`。
3. 在该条图片描述中明确目标比例、横竖方向和安全构图。
4. 禁止生成“任意比例封面图”后再在 HTML 中用 `object-fit: cover` 裁切。

封面图 prompt 必须包含：

```text
用于互动课件第 1 页封面，目标图片槽位：<slotId>，目标尺寸约 <targetPx>，目标比例 <aspectRatio>，<promptHint>。画面无文字、无标题、无公式、无页码、无边框；主体完整落在槽位安全区内。
```

常用槽位提示：

| slotId | 适用版式 | prompt 必写比例 |
|---|---|---|
| `cover-full-bleed` | `cover-01` 至 `cover-04` 整页背景 | `16:9 横向整页背景图，目标 960×540，主体避开文字安全区` |
| `cover-05-left-visual` | 左图右文分栏 | `约 0.92:1 近方形/轻竖向主视觉，目标约 499×540，主体完整` |
| `cover-06-organic-visual` | 左侧异形图右文 | `约 0.66:1 竖向主体图，目标约 326×497，适合圆角或胶囊形遮罩` |
| `cover-07-image-card` | 上文下图卡片 | `约 2.52:1 超宽横向图片卡片，目标约 900×357，不能生成竖图` |

示例：`cover-07-top-title-image-card` 的封面图描述应写成：

```text
用于互动课件第 1 页封面下方图片卡片，slotId=cover-07-image-card，目标尺寸约 900×357，目标比例约 2.52:1，超宽横向构图，鸡和兔主体横向展开且完整落在画面中央，背景简洁，适合放入白色圆角图片槽位，无文字、无标题、无公式、无页码、无边框。
```

如果一条封面图同时命中第四节内容类型，则在上述槽位提示后追加对应固定风格；如果不命中，只追加通用质量约束，不得强行套风格。

## 四、固定命中映射

每条图片描述独立判断。命中以下左侧内容类型时，必须使用右侧一一对应的固定风格，不能在命中后再自由选择其他风格。

| 左侧内容类型 | 典型图片需求 | 右侧固定风格 | 注意 |
|---|---|---|---|
| 古典文学名著 | 宫廷历史、服饰制度、器物描写、古典小说人物与场景 | 工笔/重彩（Gongbi） | 典雅、华丽、精致，适合细节丰富的古典人物与器物 |
| 古诗词、古代史 | 山水游记、人物传记、李白/苏轼等古代文人、古代历史意境 | 水墨/写意（Ink Wash） | 黑白灰、留白、虚实结合，画面飘逸、空灵、淡远 |
| 近现代文学 | 革命历史、英雄人物事迹、严肃历史叙事、近现代小说场景 | 版画/木刻（Woodblock Print） | 线条硬朗，黑白对比强烈，古朴、沧桑、有叙事力量 |
| 神话传说 | 宗教历史、丝路文化、古代艺术史、神话人物与传说场景 | 敦煌/岩彩（Dunhuang Mural） | 壁画肌理，色彩斑驳沉稳，带剥落感，神秘、古老、信仰感 |
| 传统文化 | 趣味语文、历史故事、传统节日、传统元素现代化表达 | 新国潮插画（New Guochao） | 传统元素与现代构图结合，高饱和度色彩，矢量感强，时尚、活泼、趣味 |
| 现代散文、抒情诗 | 自然景物、校园生活、亲情友情、温柔抒情场景 | 水彩淡彩（Watercolor Light） | 色彩轻柔透明，光感温和，画面干净，情绪抒情 |
| 文化交融主题 | 中外文化交流、多材料阅读写作、跨文化主题、综合性学习封面 | 拼贴艺术（Collage Art） | 纸张肌理、图形层次、低饱和配色，像文学主题视觉海报但不含文字 |
| 民俗文化、人物像（剪影） | 民俗节日、寓言故事、群像、传统故事剧场感画面 | 剪纸剪影（Paper-cut Silhouette） | 人物和场景轮廓清晰，层次分明，传统民间艺术质感 |
| 古代建筑知识 | 亭台楼阁、古建筑结构、器物纹样、传统建筑构造认知 | 白描/线稿（Bai Miao / Line Art） | 线条干净准确，主体轮廓清晰，便于课堂观察和讲解 |

匹配规则：

1. 先判断图片描述的核心教学对象属于左侧哪一类，再追加该行右侧固定风格。
2. 禁止先想用某种右侧风格，再反向改写图片类型来命中。
3. 若一条图片同时像多个类型，以图片主要教学目标和页面主题为准，只选一个左侧类型。
4. 若无法确定属于左侧任一类型，则视为不命中，不追加本指南风格。

不命中示例：

- 数学几何图、数轴、面积模型、公式推导、计算步骤图。
- 物理/化学/生物实验装置结构图。
- 统计图、流程图、表格、知识结构图。
- 练习题选项、按钮图标、奖励星星、操作提示图标。
- 需要严格还原教材原图或用户上传图片的场景。

这些图片不追加本指南的风格库。若仍需生图，只加入通用质量约束：主体清晰、构图简洁、无文字、适合对应页面图片区域；若是封面图，必须适配 `coverImageSlot` 的目标比例，不要复杂边框。

## 五、固定追加提示词

命中后，按第四节映射表追加对应提示词。不要把“工笔 + 水墨 + 敦煌 + 剪纸 + 新国潮”同时塞进同一条提示词，也不要把某一行的右侧风格挪给另一行内容类型。

### 5.1 工笔/重彩（Gongbi）

对应左侧类型：古典文学名著。

固定追加：

```text
工笔/重彩风格，线条细腻工整，色彩艳丽而典雅，朱红、石青等传统设色，人物服饰、器物和场景细节丰富，典雅、华丽、精致，无文字
```

### 5.2 水墨/写意（Ink Wash）

对应左侧类型：古诗词、古代史。

固定追加：

```text
水墨/写意风格，黑白灰为主，留白充足，笔触飘逸，虚实结合，画面飘逸、空灵、淡远，有古典文学意境，无文字
```

### 5.3 版画/木刻（Woodblock Print）

对应左侧类型：近现代文学。

固定追加：

```text
版画/木刻风格，线条硬朗，黑白对比强烈，刻痕质感明显，画面古朴、沧桑，具有力量感、叙事感和历史厚度，无文字
```

### 5.4 敦煌/岩彩（Dunhuang Mural）

对应左侧类型：神话传说。

固定追加：

```text
敦煌/岩彩风格，模仿壁画肌理，矿物色彩斑驳沉稳，带有剥落感，线条流动，画面神秘、古老、有信仰感，无文字
```

### 5.5 新国潮插画（New Guochao）

对应左侧类型：传统文化。

固定追加：

```text
新国潮插画风格，传统元素与现代构图结合，高饱和度色彩，矢量感强，画面时尚、活泼、有趣味，适合课堂课件，无文字
```

### 5.6 水彩淡彩（Watercolor Light）

对应左侧类型：现代散文、抒情诗。

固定追加：

```text
水彩淡彩风格，色彩轻柔透明，光感温和，画面干净，情绪抒情，适合语文阅读课件，无文字
```

### 5.7 拼贴艺术（Collage Art）

对应左侧类型：文化交融主题。

固定追加：

```text
拼贴艺术风格，纸张肌理和图形层次，低饱和配色，构图有节奏，画面像文学主题视觉海报，但不包含文字
```

### 5.8 剪纸剪影（Paper-cut Silhouette）

对应左侧类型：民俗文化、人物像（剪影）。

固定追加：

```text
剪纸剪影风格，人物和场景轮廓清晰，层次分明，具有传统民间艺术质感，画面简洁有戏剧感，无文字
```

### 5.9 白描/线稿（Bai Miao / Line Art）

对应左侧类型：古代建筑知识。

固定追加：

```text
白描/线稿风格，线条干净准确，主体轮廓清晰，少量淡彩，便于课堂观察和讲解，适合古建筑结构和器物纹样认知，无文字
```

## 六、模板关系

模板选择不参与“是否命中本指南”的判断。

1. `selectedTemplateId` 不能把非命中图片强行变成命中图片。
2. `selectedTemplateId` 不能把语文历史图片强行改成像素风、复古海报风或模板截图风。
3. 模板只要求图片素材具备通用承载性：主体清晰、边缘干净、背景不过度复杂、无自带文字、无厚重边框、适合放入对应页面图片区域；封面图必须适配 `coverImageSlot`。
4. 模板替换阶段只能保护并呈现已有图片 URL，不能重新生图、替换图片、给图片加滤镜或改变图片颜色。

---

## 七、调用前自检

调用 `generate_image` 前，逐条自检：

- [ ] 这条图片是模型已经判断需要生图的图片，不是为了套风格新增的图片。
- [ ] 若这是第 1 页封面图，已先确定 `data-cover-layout`、`data-cover-visual` 和 `coverImageSlot`，并把目标尺寸、比例、横竖方向写进该条 prompt。
- [ ] 已判断是否命中左侧内容类型；命中才按固定映射追加右侧风格，不命中没有套风格库。
- [ ] 命中时没有自由换风格，没有堆叠多个传统美术风格。
- [ ] 提示词使用当前生图工作流更稳定的语言；豆包工作流下优先中文。
- [ ] 没有要求图片生成可见文字、标题、题目、公式或按钮文案。
- [ ] 同一人物或同一场景的连续图片保持角色设定一致。
- [ ] 没有替换用户上传图片、教材原图或已确定使用的素材 URL。
