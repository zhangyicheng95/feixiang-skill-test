## AI 在 Phase 2.5 的操作步骤

1. 读取本文件，从上表获取所有可用模板
2. 结合大纲学科推荐模板
3. 调用 `ask_user` 单选模板 + 保底项「不使用模板，AI 自由发挥」
4. 记录用户选定的模板 ID 或 `"none"`

---

## AI 在 Phase 4 的模板装载顺序（仅当选定了某个模板时）

```
1. templates/<模板路径>/template.md        ← 风格说明书（必读）
2. templates/<模板路径>/tokens.json        ← 颜色/字体 token（参考）
3. templates/<模板路径>/shared.css         ← 必须【完整】写入 page-shared（含全部版式 CSS）
4. templates/<模板路径>/layouts/*.html     ← 按页面用途复制整块 page-data（含 exercise 的 script）
```

**当前模板包不包含**：`svg-snippets.md`、`svg/`、`examples/`。不要尝试读取这些路径。

---

## 模板的工作模型

```
┌─ 模板【管】的事 ─────────────────────────────────┐
│  ✅ HTML 版面骨架（layouts/）                    │
│  ✅ CSS 配色 / 字体 / 版式（shared.css 一份搞定） │
└────────────────────────────────────────────────┘
┌─ 模板【不管】的事 ───────────────────────────────┐
│  ❌ generate_image 画风                          │
│  ❌ 壳框架 JS（由 html-guide §3.1 固定 URL 提供） │
└────────────────────────────────────────────────┘
```

详细规则见 `html-guide.md`「〇、关于模板的硬性规则」。
