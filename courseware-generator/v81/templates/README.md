# 互动课件模板索引

> 本文件由 SKILL 的 Phase 2.5 读取，列出当前所有可用的视觉模板。AI 根据下方表格调用 `ask_user`，让用户单选模板，再据此装载模板资源进入 Phase 4。

---

## 可用模板清单

| ID | 中文名 | 学科适配 | 风格关键词 | 路径 |
|----|------|---------|----------|------|
| `math-explorer` | 数学探索风 | 初中/高中数学（含物理、化学等理科）| 学术现代、深蓝+暖橙、网格背景、几何 SVG 装饰 | `templates/数学探索风/` |

> 后续将扩展更多学科模板（语文、英语、历史等），每个模板都遵守相同结构。

---

## AI 在 Phase 2.5 的操作步骤

1. 读取本文件，从上表获取所有可用模板
2. **结合大纲的学科**给用户推荐适合的模板（如数学课件 → 推荐 `数学探索风`）
3. 调用 `ask_user`，把每个模板做成一个单选选项 + **保底项**：
   - `数学探索风（学术现代风，适合数理化学科）`
   - `不使用模板，AI 自由发挥`
4. 设置 `allowCustomAnswer=False`
5. 记录用户选定的**模板 ID**（如 `math-explorer`）或 `"none"`
6. 把选定结果传递到 Phase 3 和 Phase 4

---

## AI 在 Phase 4 的模板装载顺序（仅当选定了某个模板时）

按以下顺序读取选定模板下的入口文件：

```
1. templates/<模板路径>/template.md        ← 风格说明书（必读）
2. templates/<模板路径>/tokens.json        ← 颜色/字体/间距 token
3. templates/<模板路径>/shared.css         ← 必须完整写入 <template class="page-shared"> 的 <style>
4. templates/<模板路径>/layouts/*.html     ← 按页面用途选用版式骨架
5. templates/<模板路径>/svg-library.md     ← SVG 图标库索引
6. templates/<模板路径>/svg/*.svg          ← 矢量图标库（AI 主动挑选 inline）
7. templates/<模板路径>/examples/sample.html  ← 完整样张（视觉参考）
```

## 模板的工作模型

```
┌──────────────────────────────────────────────────────────┐
│  模板【管】的事                                            │
├──────────────────────────────────────────────────────────┤
│  ✅ HTML 版面骨架（layouts/）                              │
│  ✅ CSS 配色 / 字体 / 间距（shared.css）                   │
│  ✅ SVG 装饰图标库（svg/）                                 │
└──────────────────────────────────────────────────────────┘
┌──────────────────────────────────────────────────────────┐
│  模板【不管】的事                                          │
├──────────────────────────────────────────────────────────┤
│  ❌ generate_image 的画风（AI 完全自由生图）               │
│  ❌ 课件内容插图（如鸡兔同笼的鸡兔形象）                   │
│  ❌ 互动逻辑实现（saveState/restoreState）                 │
└──────────────────────────────────────────────────────────┘
┌──────────────────────────────────────────────────────────┐
│  AI 的自由度                                              │
├──────────────────────────────────────────────────────────┤
│  ✅ 主动从 svg/ 库挑 1-2 个 SVG inline 到页面装饰          │
│  ✅ 微调元素位置、大小、间距以适应内容长短                  │
│  ✅ 调整图片尺寸（image-slot 用 object-fit:contain）       │
│  ✅ 增减版式内的元素数量（如 concept 的 3 卡 → 2 卡）       │
├──────────────────────────────────────────────────────────┤
│  AI 必须保证                                              │
├──────────────────────────────────────────────────────────┤
│  ❌ 元素不能重叠（图片不能盖文字、装饰不能盖内容）           │
│  ❌ 内容不能溢出 960×540 画布                              │
│  ❌ 配色字体严格走 CSS 变量                                 │
└──────────────────────────────────────────────────────────┘
```

详细规则见 `html-guide.md` 中"〇、关于模板的硬性规则"章节。
